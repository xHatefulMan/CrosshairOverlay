# CrosshairOverlay

A lightweight C# WinForms overlay that displays a fully customizable crosshair on top of any game or application.

> The application interface is in French.

## Download

👉 [Latest release (v1.0)](https://github.com/xHatefulMan/CrosshairOverlay/releases/tag/v1.0)

> No installation required — just download and run the `.exe`

---

## Features

### 6 crosshair styles
- **Cross** — classic crosshair
- **Dot** — single center dot
- **Cross + Dot** — cross with center dot
- **CS Gap** — Counter-Strike style crosshair with adjustable center gap
- **Circle** — circular crosshair
- **Circle + Cross** — circle with cross overlay

### Customization
- 8 preset colors + full **custom color picker**
- Adjustable **size**, **thickness**, and **opacity** via sliders
- **Gap Centre** slider (CS Gap style only)

### Hotkey
- Toggle the crosshair on/off with a customizable hotkey (default: `F2`)
- Change the hotkey at any time by pressing any key in the hotkey dialog

### Auto-launch by game
- Automatically enables the crosshair when a predefined game is detected

### Options
- Show crosshair on startup
- Launch app at Windows startup
- Settings saved between sessions

---

## Usage

1. Download `CrosshairOverlay.exe` from the [Releases](https://github.com/xHatefulMan/CrosshairOverlay/releases) page
2. Run it — no installation needed
3. Choose your crosshair style, color and size
4. Press `F2` (or your custom hotkey) to toggle the crosshair on/off
5. Click **Save** to keep your settings
