# CrosshairOverlay
Crosshair Overlay est un overlay WPF C# proposant 6 styles de crosshair ajustables en couleur, taille, épaisseur et opacité. Un raccourci clavier personnalisable permet de l'activer ou le désactiver. L'application peut se lancer en arrière-plan au démarrage de Windows et activer automatiquement le crosshair à l'ouverture d'un jeu prédéfini.
